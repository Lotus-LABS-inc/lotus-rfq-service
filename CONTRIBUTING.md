# Contributing

Lotus RFQ Service is a proprietary core infrastructure repository. External
contributions are not accepted unless Lotus has approved the contribution in
writing before submission.

Before submitting work, contributors must have a written contributor agreement,
CLA, contractor agreement, employment agreement, or IP assignment approved by
Lotus. By submitting any contribution, the contributor confirms that they have
the full right to submit the work and that the work does not violate any third
party rights.

Do not copy third-party proprietary code, confidential code, leaked code, or
code from sources whose license is incompatible with Lotus' proprietary use.
Generated code from restricted, confidential, or unclear sources must not be
added without legal and engineering review.

Unless Lotus agrees otherwise in writing, all accepted contributions become part
of the proprietary Lotus codebase and are governed by this repository's
proprietary license.
