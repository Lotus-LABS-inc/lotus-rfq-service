# Generated Markdown Layout

Generated markdown reports do not belong flat at the `docs/` root.

Use:

- `docs/generated/crypto/`
- `docs/generated/sports/`
- `docs/generated/politics/`
- `docs/generated/shared/`

Rules:

1. Generated markdown companions, operator summaries, audits, and report outputs belong here.
2. Human-authored docs belong in `docs/design/`, `docs/runbooks/`, `docs/alerts/`, and `docs/delivery/`.
3. The top-level `docs/` folder should stay reserved for a small number of stable navigation docs such as `ARCHITECTURE.md`.
